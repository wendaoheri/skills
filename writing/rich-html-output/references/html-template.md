# HTML Document Template

Use this template for every document generated by the rich-html-output skill. Replace `{{PLACEHOLDER}}` values. The CSS is self-contained — no external dependencies except the optional CDN scripts for syntax highlighting (highlight.js) and diagram rendering (Mermaid.js). Both degrade gracefully offline.

## Placeholder Reference

| Placeholder | What to fill in |
|---|---|
| `{{LANG}}` | `zh-CN` for Chinese docs, `en` for English |
| `{{TITLE}}` | Full document title |
| `{{DATE}}` | ISO date: `2026-05-24` or `2026年5月24日` |
| `{{DOC_TYPE}}` | e.g. `架构设计`, `技术选型`, `需求分析`, `调研报告` |
| `{{SECTION_N_ID}}` | kebab-case anchor, e.g. `background`, `system-design` |
| `{{SECTION_N_TITLE}}` | Human-readable heading text |

## Component Usage Guide

**Callout variants:**
```html
<div class="callout">           <!-- info (blue) -->
<div class="callout warning">   <!-- warning (yellow) -->
<div class="callout danger">    <!-- danger (red) -->
<div class="callout success">   <!-- success (green) -->
```

**Badge variants:**
```html
<span class="badge">default</span>
<span class="badge green">推荐</span>
<span class="badge yellow">待定</span>
<span class="badge red">不推荐</span>
<span class="badge gray">已废弃</span>
```

**Mermaid diagrams** (renders as SVG — use instead of ASCII art):
```html
<!-- Architecture diagram -->
<div class="mermaid">
flowchart LR
  subgraph 客户端
    App[Web App]
  end
  subgraph 服务层
    GW[API Gateway]
    Auth[认证服务]
  end
  subgraph 存储层
    Redis[(Redis)]
    DB[(PostgreSQL)]
  end
  App -->|HTTPS| GW
  GW --> Auth
  Auth --> Redis
  Auth --> DB
</div>

<!-- Sequence diagram -->
<div class="mermaid">
sequenceDiagram
  participant C as Client
  participant G as API Gateway
  participant A as Auth Service
  C->>G: POST /auth/login
  G->>A: validateCredentials()
  A-->>G: JWT token
  G-->>C: 200 OK + token
</div>

<!-- ER diagram -->
<div class="mermaid">
erDiagram
  USER ||--o{ ORDER : places
  ORDER ||--|{ ORDER_ITEM : contains
  ORDER_ITEM }o--|| PRODUCT : refers-to
</div>
```

**Decision record block** (for ADR-style docs):
```html
<div class="decision">
  <div class="decision-header">决策：选择 PostgreSQL</div>
  <div class="decision-body">
    <p><strong>背景：</strong>...</p>
    <p><strong>决策：</strong>...</p>
    <p><strong>后果：</strong>...</p>
  </div>
</div>
```
