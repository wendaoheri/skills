# 目标澄清参考

在目标路径设计流程的第一阶段，需要帮助用户明确他们真正想要达成的目标以及背后的动机。此文件提供目标分类、关键提问、目标质量判断等工具，用于结构化地澄清目标。

## 目标类型

在对话开始时，首先判断用户的目标属于哪一类。常见类型包括：

1. **职业发展目标**：升职、跳槽、技能提升、建立行业影响力等。
2. **创业或产品目标**：研发新产品、建立公司、商业化某一创意等。
3. **学习成长目标**：完成学位、掌握新技能、通过考试等。
4. **财富或投资目标**：实现财务自由、投资某个领域、增加收入等。
5. **关系与家庭目标**：改善人际关系、组建家庭、陪伴孩子成长等。
6. **健康与生活方式目标**：运动习惯、减重、戒烟戒酒、改善睡眠等。
7. **个人影响力目标**：建立社交媒体影响力、发表作品、参与公共事务等。
8. **组织或团队目标**：推动团队达成业务目标、改善协作流程等。
9. **迁移、身份或生活选择**：移民、换城市、申请身份、重新规划生活等。
10. **长期人生方向目标**：探索人生意义、追求幸福感、实现长期使命等。

识别目标类型有助于选用合适的方法和指标，也能提醒用户检查目标是否是自己的真实意愿而非外界压力。

## 目标澄清问题

使用以下关键问题挖掘目标背后的真实动机和预期结果。每次提问不超过 5 个，根据对话情况灵活选择：

1. 你想达成的具体结果是什么？它在多大程度上是外部可观察的？
2. 你希望在什么时间范围内达成这个目标？这个时间是否现实？
3. 为什么这个目标现在对你重要？它解决了你生活或工作中的什么问题？
4. 如果达成这个目标，你的生活、事业或情绪会发生什么改变？最重要的收益是什么？
5. 如果没有达成，会出现什么后果？最坏的情况是什么？
6. 这个目标是你真正想要的，还是你觉得自己“应该”要的？你如何区分两者？
7. 你愿意为这个目标投入哪些资源（时间、精力、金钱）？你绝对不愿意牺牲什么？
8. 是否有人或某些环境因素推动你设定这个目标？如果是，这些因素对你有多大影响？

通过这些问题，可以帮助用户剥离表面的愿望，理解目标背后的动机、期望和约束。

## 目标定义模板

澄清目标后，可以用以下格式将其转化为清晰的目标描述：

```
我希望在【时间范围】内，通过【主要路径】达成【具体结果】，以便解决【真实问题】或获得【核心收益】。
```

例如：

> 我希望在未来 6 个月内，通过系统学习和实践，设计并发布一个 AI Agent 产品原型，以便提升自己的产品能力并在公司内部获得认可。

## 目标质量判断

一个优质的目标通常具备以下特点：

- **结果具体**：目标描述清晰，别人可以观察到是否达成。
- **时间边界清楚**：有明确的截止日期或时间范围。
- **成本可承受**：达成目标所需的投入在自身能力范围内，不会引发重大负担。
- **动机真实**：目标源于个人真实愿望和长期动机，不是短期焦虑或外界压力驱动。
- **可被验证**：有衡量目标达成程度的指标或客观标准。
- **可拆分**：可以分解为阶段性小目标和行动步骤。

## 警惕的目标类型

在澄清目标时，还需要警惕以下情况：

- **过于宏大，但缺乏路径**：如“改变世界”“赚很多钱”，没有具体方案和时间表。
- **看似理性，实则逃避现状**：如用继续学习逃避行动，用不断规划替代执行。
- **由焦虑驱动**：目标背后是短期压力或外界比较，没有真正的内在动机。
- **只描述身份，不描述行动**：如“成为行业大佬”，却没有具体行动方案。
- **只描述结果，不考虑代价**：忽视达成目标所需的努力和资源投入。

通过识别上述风险，可以帮助用户避免设定不切实际或无效的目标，为后续的计划奠定坚实基础。
