# 背景挖掘参考

在目标路径规划中，充分了解用户的个人背景是制定可行方案的关键。此文件提供挖掘用户资源、限制和行为模式的框架，以及示范问题和输出模板。

## 背景信息类别

收集以下类别的信息有助于判断用户可利用的资源和必须考虑的约束：

1. **当前身份与角色**：职业、职位、家庭角色、社交身份等。
2. **工作背景与专业能力**：行业经验、硬技能、软技能、专业证书或学位等。
3. **已有资源与资产**：人脉、资金、工具、数据、平台、品牌影响力等。
4. **时间和精力约束**：每日/每周可用时间、精力波动规律、健康状况等。
5. **家庭和生活约束**：家庭责任、育儿或照顾父母、伴侣支持度、居住环境等。
6. **财务承受能力**：储蓄、收入来源、投资能力、风险承受度。
7. **性格与行为模式**：是否偏向自律还是容易拖延，喜欢探索还是遵循计划，独立推进还是需要监督等。
8. **过往成功与失败经验**：历史项目或目标完成情况，成功模式和经常遇到的阻碍。
9. **当前最大的压力来源**：工作压力、家庭压力、健康问题、经济压力等。

## 背景挖掘问题

在与用户交互时，可以使用以下问题探究上述信息类别。根据需要灵活选择 3–5 个问题：

1. 你现在的工作或生活状态是什么？有哪些角色需要同时兼顾？
2. 你每周能够稳定投入多少时间和精力用于该目标？有没有时段是效率高峰或低谷？
3. 你有哪些可以利用的资源（人脉、技能、设备、资金、信息）？哪些是你过去忽视但有价值的？
4. 你最擅长做什么，最不擅长什么？过去在哪些方面容易遇到瓶颈？
5. 你之前做过类似的事吗？成功的原因是什么？失败通常卡在什么地方？
6. 目前有哪些家庭或生活方面的约束，例如照顾家人、不可改变的日程安排等？
7. 你的健康状况和精神状态如何？需要注意什么潜在问题？
8. 你的财务状况是否允许你投入一定的资金或时间？你的风险承受度如何？
9. 你更适合高强度冲刺还是长期稳定推进？你更需要外部监督还是可以自我约束？
10. 你现在最大的压力来源是什么？这个压力会如何影响你达成目标？

## 输出模板示例

在整理背景信息时，可以采用如下结构：

```
### 已知背景
- 工作与角色：……
- 专业能力：……
- 资源与资产：……
- 时间与精力：……
- 家庭与生活：……
- 财务承受能力：……
- 行为模式与性格：……
- 过往经验：……
- 当前压力：……

### 关键资源
- …

### 关键约束
- …

### 个人执行特征
- …

### 仍需确认的信息
- …
```

通过系统地梳理这些信息，可以让目标规划更贴近用户实际，避免设计超出用户能力或忽视关键约束的计划。
